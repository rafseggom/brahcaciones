<task type="auto">
  <name>Task 1: Auth Context & Persistence</name>
  <files>src/context/AuthContext.tsx</files>
  <action>
    Create a React Context to manage authentication state:
    - State: `userSlug` (string | null), `isAdmin` (boolean), `isAuthenticated` (boolean).
    - On mount: Check `localStorage` for `brahcaciones_slug`.
    - Function `login(code: string)`: 
      - Compares `code` against values in `import.meta.env` (VITE_AUTH_MOLTEN, VITE_AUTH_KEVIROLO, VITE_AUTH_WARGIOS, VITE_AUTH_GONPAR, VITE_AUTH_DEKKER, VITE_AUTH_ADMIN).
      - If valid, sets `userSlug` (the name part of the env var, e.g., 'molten' or 'admin') and saves it to `localStorage`.
    - Function `logout()`: Clears state and `localStorage`.
    Export `AuthProvider` and `useAuth` hook.
  </action>
  <verify>
    <automated>grep -q "localStorage" src/context/AuthContext.tsx && grep -q "VITE_AUTH_" src/context/AuthContext.tsx</automated>
  </verify>
